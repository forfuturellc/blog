A simple, super-powered file-pushing process

GochoMugo http://www.gmugo.in GochoMugo Sep 22, 2016
  • software architecture
  • tools

This post introduces fpush, a library and terminal tool for pushing files to remote servers over multiple protocols, including (at time of writing) FTP and SFTP. The library utilizes a push process designed to be interruptible (at a minimal cost) and protocol-agnostic.
